# ChPio — Chat Pioneer

Self-hosted AI workspace. Chat with any local model or API, organize with projects, and take notes — all local-first, privacy-first.

## Features

- **Chat** — OpenRouter, OpenAI, Ollama, llama.cpp, Groq, DeepSeek, Mistral, and more
- **Projects** — organize chats into folders with skills and special instructions
- **Notes & Tasks** — quick notes with reminders (coming soon)
- **Documents** — multi-tab editor with AI assist (coming soon)
- **Deep Research** — multi-step research with visual reports (coming soon)
- **Calendar** — local-first with CalDAV sync (coming soon)
- **Memory** — persistent memory and skills (coming soon)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173), go to Settings, add your API key, sync models, and start chatting.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Zustand
- Framer Motion
- Lucide Icons
