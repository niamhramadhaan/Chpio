# Notes System — Archive + Toolbar Redesign

## Goal
Add archive system (matching chat) for folders and notes, redesign toolbar with chip-style buttons, search expand, and extend menu.

## Files to Modify
1. `src/types/index.ts` — add archive fields
2. `src/store/notesStore.ts` — add archive/unarchive actions
3. `src/pages/NotesPage.tsx` — full UI rewrite

---

## 1. Types (`src/types/index.ts`)

Add `archived` and `archivedAt` to both interfaces:

```ts
export interface NoteFolder {
  id: string;
  name: string;
  archived: boolean;
  archivedAt?: number;
  createdAt: number;
  updatedAt: number;
}

export interface Note {
  id: string;
  title: string;
  content: string;
  tasks: NoteTask[];
  folderId: string;
  archived: boolean;
  archivedAt?: number;
  createdAt: number;
  updatedAt: number;
}
```

## 2. Store (`src/store/notesStore.ts`)

Add to state interface:
- `archiveFolder(id)` — sets `archived: true, archivedAt: Date.now()`
- `unarchiveFolder(id)` — sets `archived: false, archivedAt: undefined`
- `archiveNote(id)` — same pattern
- `unarchiveNote(id)` — same pattern
- `getArchivedFolders()` — returns `folders.filter(f => f.archived)`
- `getArchivedNotes(folderId?)` — returns archived notes, optionally filtered by folder
- Update `getFolderNotes` to exclude archived notes
- Update `deleteFolder` to also delete associated notes

## 3. NotesPage UI (`src/pages/NotesPage.tsx`)

### 3a. FolderCard — Archive instead of Delete
- Replace trash icon with Archive icon
- Inline confirm: "Archive" (amber) + "Cancel" buttons
- Match chat pattern exactly

### 3b. NoteCard — Add Archive + Delete
- Two action buttons on hover: Archive (amber) + Delete (red)
- Archive shows inline confirm: "Archive" + "Cancel"
- Delete shows inline confirm: "Delete" + "Cancel"

### 3c. Bottom Toolbar — Chip-style with Search + Extend
New design:
- **Container**: `bg-[#0f1413]/60 backdrop-blur-md rounded-xl mx-3 mb-3 border border-white/5`
- **Layout**: Horizontal row of chips
- **Search chip**: Left side, Search icon. When clicked, expands inline (animates other chips left) with a text input
- **Action chips**: Folder/Note, Import, Select — pill-shaped with `bg-white/5 backdrop-blur-sm border border-white/5 rounded-full`
- **Extend chip**: Right side, `MoreHorizontal` icon with `>` chevron. Clicking opens a dropdown with more options (Archived, Settings, etc.)
- **Chips style**: `px-3 py-1.5 rounded-full text-[10px] font-medium bg-white/5 border border-white/5 hover:bg-white/10 hover:border-white/10`

### 3d. Archived View
- When "Archived" is clicked from extend menu, show archived folders/notes
- Back button returns to normal view
- Unarchive button on each archived item (same inline pattern as chat)
